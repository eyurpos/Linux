# Useful commands
- Count lines
  > wc -l 
- Print in shell and file
  > tee
- Sorting
  > sort
- Unique entety
  > uniq (should be sorted)
- Replace char
  > tr 'a' 'b'
- Delete char
  > tr -d ''
- Cut bytes
  > cut
- Shell features
  > set ('-' - enable, '+' - disable feature)
- Disk usage
  > du 
- File transfer remote
  > sftp

  > scp [user]@[host]:[source_path] [user]@[host]:[dest_path]
